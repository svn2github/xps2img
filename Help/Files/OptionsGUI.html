GUI Frontend Command-line Options

Batch Mode

-b, --batch

Starts GUI Frontend in batch mode.

Features:

  • Application is started minimized.
  • All CLI options are supported.
  • Error messages are printed to parent process console.
  • On success, error or cancel/close application returns corresponding exit code.
  • Copy and use batch command-line in batch script as is.

Image Prefix Incudes Document Name

Image prefix incudes document name: src_file-01.png

Image DPI Instead Of Size

Use image DPI instead of size.

Processors

--processors-number=number

Number of simultaneously running document processors.

Number of logical CPUs by default (Auto)

One of:

  • auto
  • up to number of logical CPUs
