GUI Command-line Options

Processors

--processors-number=number

Number of simultaneously running document processors.

Number of logical CPUs by default (Auto)

One of:

  • Auto
  • Up to number of logical CPUs.

Processors Priority

--processors-priority=priority

Document processors priority.

Normal by default (Auto)

One of:

  • Auto
  • Idle
  • Below Normal
  • Normal
  • Above Normal
  • High
