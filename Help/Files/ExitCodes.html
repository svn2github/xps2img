Exit Codes

Returned by both CLI and GUI Frontend.

Code Description
0 Conversion succeeded.
1 Invalid command-line arguments.
2 No command-line arguments were specified. Help is displayed.
3 Conversion failed.
4 Pages specified are out of document pages range.
5 Conversion was aborted by user.

Batch Script Usage Example

Prints exit code corresponding message. 

@echo off
if %errorlevel% lss 0 echo Unknown exit code.
if %errorlevel% equ 0 echo Conversion succeeded.
if %errorlevel% equ 1 echo Invalid command-line arguments.
if %errorlevel% equ 2 echo No command-line arguments were specified. Help is displayed.
if %errorlevel% equ 3 echo Conversion failed.
if %errorlevel% equ 4 echo Pages specified are out of document pages range.
if %errorlevel% equ 5 echo Conversion was aborted by user.
if %errorlevel% geq 6 echo Unknown exit code.
