Command-line Options

Mandatory arguments to long options are mandatory for short options too.

src-file

XPS file to process. Required.

[out-dir]

Output folder. Optional.

current folder by default.

-p, --pages=pages

Page number(s).

all pages by default.

1-
all
1
single
1,3
set
1-10 or -10 or 10-
range
1,3-5,7-9,15-
combined

-f, --file-type=type

Image type.

png by default.

One of:

  • png
  • jpeg
  • tiff
  • bmp
  • gif

-r, --required-size=size

Desired image size.

DPI will be ignored if specified.

width only
2000
height only
x1000
both
2000x1000
width for landscape orientation.
height for portrait orientation.

-d, --dpi=dpi

Image DPI (16-1500).

120 by default.

-i, --image-name=name

Image prefix.

numeric if ommited
01.png
name of src file if empty (-i "")
src_file-01.png

-q, --jpeg-quality=quality

JPEG quality level (10-100).

85 by default.

-t, --tiff-compression=compression

TIFF compression method.

zip by default.

One of:

  • default
  • none
  • ccitt3
  • ccitt4
  • lzw
  • rle
  • zip

-a, --first-page-index=index

Document body first page index.

1 by default.

-x, --prelims-prefix=prefix

Preliminaries prefix.

$ by default.

-e, --test

Test mode (no files will be written).

-s, --silent

Silent mode (no progress will be shown).
