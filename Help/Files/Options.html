Command-line Options

Usage

xps2img src-file [out-dir] [options]

Mandatory arguments to long options are mandatory for short options too.

See also exit codes.

XPS Document

src-file

XPS document to process. Required.

Output Folder

[out-dir]

Output folder. Optional.

new folder named as document will be created in folder where document is by default.

After Conversion

-w, --post-action=action

Action to execute after conversion completed. Only available actions are shown.

One of:

  • donothing
  • shutdown
  • reboot
  • sleep
  • hibernate
  • logoff
  • exit

If conversion fails on startup error message will be shown and no action executed. Active only during conversion session.

Image Type

-f, --file-type=type

Image type.

png by default.

One of:

  • png
  • jpeg
  • tiff
  • bmp
  • gif

Specify this option to shorten image extension down to three characters.

JPEG Quality

-q, --jpeg-quality=quality

JPEG quality level (10-100)

85 by default.

TIFF Compression

-t, --tiff-compression=compression

TIFF compression method.

zip by default.

One of:

  • none
  • ccitt3
  • ccitt4
  • lzw
  • rle
  • zip

CCITT G3, CCITT G4 and RLE produce black and white images.

Image Size

-r, --required-size=size

Desired image size (greater or equal 10)

DPI will be ignored if image size is specified.

width only

height only
x1000

Image DPI

-d, --dpi=dpi

Image DPI (16-2350)

120 by default.

DPI will be ignored if image size is specified.

Image Prefix

-i, --image-name=name

Image name prefix. <>:"/\|?* characters are not allowed.

numeric if omitted
01.png
name of document if empty (-i "")
document-01.png

First Page Index

-a, --first-page-index=index

Document body first page index or desired and actual page numbers in any order.

1 by default.

5
numeric
"96 100" or 96-100
desired and actual page numbers

Preliminaries Prefix

-x, --prelims-prefix=prefix

Preliminaries prefix. <>:"/\|?* characters are not allowed.

$ by default.

Shorten Image Extension

-n, --shorten-extension

Shorten image extension down to three characters.

  • normal
  • abovenormal
  • high

Ignore Existing Images

-k, --ignore-existing

Do not overwrite existing images.

Will be reset in case of any image related property change.

Ignore Conversion Errors

-g, --ignore-errors

Ignore conversion errors.

Silent

-s, --silent

Silent mode (no progress will be shown)

Test Mode

-e, --test

Test mode (no files will be written)

Clean

--clean

Clean (delete images)
