XPS Conversion

  • Use PNG Image Type for better quality and fastest processing.
  • Image DPI and Image Size are mutually exclusive; the latter has higher priority.
  • First Page Index is the number of document body first page. Useful for images names synchronization with book page numbers.
  • Document processors speedup document processing up to number of CPUs times.
  • If conversion could not be performed due to insufficient memory try to reduce Processors / Image DPI / Image Size.
  • Ignore Existing Images skips page conversion if image already exists.
  • Ignore Conversion Errors makes conversion completed regardless of errors. Failed pages will be reported at the end.
  • Use Convert popup menu item to restart conversion regardless of resume status.
  • All CLI/GUI command-line options can be used with GUI Frontend.
