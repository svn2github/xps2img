XPS Conversion

  • Use Png Image Type for better quality.
  • Image DPI and Image Size are mutually exclusive; the latter has higher priority.
  • First Page Index is the number of document body first page. Useful for synchronizing images names with book page numbers.
  • Processors speedups document processing up to number of CPUs times.
  • If your system lacks resources use single processor.
  • Use Convert popup menu item to restart conversion regardless of resume status.
  • All CLI/GUI command-line options can be used with GUI Front-end.
