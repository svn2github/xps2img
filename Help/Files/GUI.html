GUI Frontend

Features

  • Serves as frontend for xps2img command-line utility.
  • Fully supports xps2img CLI/GUI command-line.
  • Supports batch mode.
  • Allows to resume conversion (only if no options were changed and/or GUI options were alerted):

    Use Convert popup menu item to restart conversion regardless of resume status.

  • Allows converted images removal:
  • Post conversion action support.
